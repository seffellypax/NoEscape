# NO ESCAPE 🎮

## 📌 Project Description

NO ESCAPE is an interactive fiction/visual novel game that places the player in the role of a student trapped in an endless campus time loop. Every decision matters as the player explores different locations, uncovers crucial clues, and navigates challenges in a race to graduate and break free from the cycle.

## 🎮 Genre

Interactive Fiction / Visual Novel

## 🧠 Core Features

* Scene-based storytelling
* Player decision-making (choices)
* Branching paths
* Clue/Inventory system
* Multiple outcomes (win/loop)
